# function to return the data behind where the user has clicked

function to return the data behind where the user has clicked

## Usage

``` r
click_info(data, click_x, facet = NULL)
```

## Arguments

- data:

  the data underpinning the chart that has been clicked

- click_x:

  the x location where the click occurred

- facet:

  the months_waited_id that the data was selected from when the plots
  are faceted
